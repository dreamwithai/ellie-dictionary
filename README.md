# 📚 윤채의 단어장

중학생을 위한 영어 단어장 웹앱입니다.

## ✨ 주요 기능

- 📖 **단어장 생성 및 관리**
- ➕ **영어-한글 단어 추가/수정/삭제**
- 🔍 **단어 검색 기능**
- 👁️ **단어 숨기기/보이기 (암기 연습)**
- ✅ **암기 완료 체크 기능**
- 🎯 **단어 테스트 기능**
- 💾 **데이터 백업/복원**
- 📱 **모바일 최적화**

## 🚀 사용법

1. 웹사이트 접속
2. "새 단어장 만들기" 클릭
3. 단어장 제목과 설명 입력
4. 영어 단어와 한글 뜻 추가
5. 다양한 학습 기능 활용

## 💾 데이터 저장

- 모든 데이터는 브라우저에 자동 저장됩니다
- 백업 기능으로 데이터를 파일로 내보낼 수 있습니다
- 오프라인에서도 사용 가능합니다

## 🛠️ 기술 스택

- React 18
- Lucide React (아이콘)
- Webpack
- CSS Grid & Flexbox
- localStorage
- Service Worker (PWA)

## 📱 모바일 지원

- 반응형 디자인
- 터치 최적화
- PWA 지원 (홈 화면에 추가 가능)
